# Frontend Next.js - TourAI Việt

Frontend đã được chỉnh theo yêu cầu mới:

- Thanh tìm kiếm ở `/tours` có thêm tìm bằng ảnh và giọng nói.
- Có tab `/assistant` để chat với AI riêng như giao diện nhắn tin.
- Admin `/admin` có wizard 3 bước cho quản lý tour.

## Chạy local

```bash
npm install
npm run dev
```

## Route chính

- `/`
- `/tours`
- `/tour/[slug]`
- `/assistant`
- `/dashboard`
- `/admin`
